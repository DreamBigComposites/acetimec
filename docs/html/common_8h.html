typedef int32_t atc_time_t

The number of seconds from the current epoch.

The current epoch is 2050-01-01 00:00:00 UTC by default, but can be adjusted using atc_set_current_epoch_year().

kAtcInternalEpochYear 

Epoch year used by the internal converter functions atc_convert_to_internal_days() and atc_convert_from_internal_days() so that the "internal epoch" is {year}-01-01T00:00:00.

This must be a multiple of 400.

This is an internal implementation detail and should not normally be needed by client applications. They should instead use atc_get_current_epoch_year() and atc_set_current_epoch_year().

kAtcDaysToInternalEpochFromUnixEpoch 

Number of days from the Unix epoch (1970) to the Internal epoch (2000).

kAtcDefaultEpochYear 

Default epoch year of the library.

This is selected so that the 32-bit atc_time_t epoch seconds type is valid at least over the 100 years from 2000 to 2100.

kAtcDaysToDefaultEpochFromInternalEpoch 

Number of days from the Internal epoch (2000) to the default epoch (2050).

This is 50 years * 365 + 13 leap days.

kAtcInvalidYear 

Sentinel value for invalid year.
