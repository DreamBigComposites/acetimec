Detailed Description

Functions that relate to the local date time along with a UTC offset measured in minutes.

For example, this can be used to represent a fixed offset from UTC, such as "2022-08-30 14:45:00-08:00".

◆ AtcOffsetDateTime

typedef struct AtcOffsetDateTime AtcOffsetDateTime

Date and time fields with a UTC offset in minutes.

In the AceTime library, the 'fold' parameter was included in the low-level LocalTime class, which then got absorbed into the LocalDatetime class. This extra parameter is mostly transparent to the user because C++ supports default parameters in the constructor and functions.

Unfortunately C does not have default parameters, so adding a 'fold' in LocalDateTime causes unnecessary friction. Therefore, I am adding this parameter to the OffsetDateTime instead, which is higher level so hopefully the parameter is exposed to the user only when the user needs.
